<details>
<summary>3가지 법칙 요약</summary> 

> 1. 브룩스의 법칙: 인원이 더 들어와도 작업시간이 빨라지진 않는다.<br/>
> 2. 파레토 법칙: 20%의 부분에서 80%의 결과가 나타난다. -> 결함과 오류의 대부분은 특정부분에서 발생한다.
> 3. 롱테일 법칙: 80%의 평범한 사람이 20%의 뛰어난 사람보다 낫다. (파레토 법칙의 반대)
